{% trans "To create a Harbor account:" %}

  2. {% trans "Fill in your personal information including your name, email address, and a username and password." %}
  3. {% trans "Accept the Terms of Service and Privacy Policy." %}
  4. {% trans "Click Create Account to complete registration." %}

{% trans "After registering, you will be able to log in and set up your organization profile. Your organization profile is required before you can submit applications." %}

{% trans "Note: State agency staff should contact their agency administrator for account provisioning rather than using self-registration." %}

{% trans "To apply for a funding opportunity:" %}

  1. {% trans "Browse available opportunities on the" %} {% trans "Funding Opportunities" %} {% trans "page." %}
  2. {% trans "Click View Details on an opportunity you are interested in." %}
  3. {% trans "Review the program description, eligibility criteria, and required documents." %}
  4. {% trans "If the program is currently accepting applications, click the Apply Now button." %}
  5. {% trans "Complete all required sections of the application form, attach supporting documents, and submit." %}

{% trans "You can save your application as a draft and return to it later before the deadline. Make sure to submit before the application deadline, as late submissions cannot be accepted." %}

{% trans "Your application may have one of the following statuses:" %}

{% trans "Draft" %}
{% trans "You have started an application but have not yet submitted it. You can continue editing until you submit." %}
{% trans "Submitted" %}
{% trans "Your application has been submitted and received by the awarding agency. It is awaiting initial review." %}
{% trans "Under Review" %}
{% trans "Your application is currently being reviewed and evaluated by the agency or review panel." %}
{% trans "Revision Requested" %}
{% trans "The agency has requested changes or additional information. Check your notifications for details on what needs to be revised." %}
{% trans "Approved" %}
{% trans "Your application has been approved for funding. An award will be processed and you will receive further instructions." %}
{% trans "Denied" %}
{% trans "Your application was not selected for funding in this cycle. You may receive feedback from the agency about your application." %}
{% trans "Withdrawn" %}
{% trans "You or the agency has withdrawn this application from consideration." %}

{% trans "Once you have an active award, you may be required to submit periodic progress and financial reports. To submit a report:" %}

  1. {% trans "Log in to your account and navigate to My Awards." %}
  2. {% trans "Select the award you need to report on." %}
  3. {% trans "Click the Reports tab to see upcoming and past-due reports." %}
  4. {% trans "Click Submit Report next to the report that is due." %}
  5. {% trans "Complete the report form with program activities, expenditure details, and any required attachments." %}
  6. {% trans "Submit the report for agency review." %}

{% trans "Report due dates are set by the awarding agency and will appear in your notifications. Timely reporting is important for continued funding and compliance with your award agreement." %}
